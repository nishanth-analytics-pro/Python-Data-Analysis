# 🎬 Netflix Content Strategy Analysis | Python EDA

## 📌 Project Overview
This project involves a deep-dive Exploratory Data Analysis (EDA) of the Netflix dataset to uncover global streaming trends. By analyzing content distribution across genres, countries, and time, this study provides a data-driven perspective on how Netflix's library has evolved over the years.

---

## 🎯 Objectives
*   **Data Integrity**: Performed rigorous cleaning, including null value removal and duplicate checks across 7,000+ records.
*   **Time-Series Analysis**: Converted raw data into datetime formats to track year-over-year content growth.
*   **Global Mapping**: Identified the top 10 content-producing nations to understand regional dominance.
*   **Genre Intelligence**: Deconstructed complex genre tags to map distribution across various categories.

---

## 💾 Dataset Information
The analysis is based on the Netflix Movies and TV Shows dataset from Kaggle.
*   **Source**: [Kaggle - Netflix Movies and TV Shows](https://www.kaggle.com/datasets/shivamb/netflix-shows)
*   **Data Size**: ~7,000+ records with 12 metadata columns.
*   **Focus**: Title, Director, Cast, Country, Date Added, and Genre.

---

## 🛠️ Technical Workflow
1.  **Data Cleaning**: Handled null values in critical columns and verified for duplicate records.
2.  **Feature Engineering**: Derived `year_added` from the `date_added` column for longitudinal trend analysis.
3.  **Visualization**: Utilized Matplotlib and Seaborn to create high-fidelity charts for year-wise trends and genre distribution.

---

## 💡 Key Business Insights
*   📈 **Year-wise trend**: Identified specific years where Netflix aggressively expanded its content library.
*   🌍 **Regional Dominance**: Mapped top 10 countries contributing to the platform's global reach.
*   🎞️ **Content Mix**: Analyzed the strategic ratio between Movies vs. TV Shows over time.
